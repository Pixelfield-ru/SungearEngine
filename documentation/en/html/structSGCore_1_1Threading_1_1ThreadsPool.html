◆ m_maxThreadsCycleExecutionPassTime

double SGCore::Threading::ThreadsPool::m_maxThreadsCycleExecutionPassTime = 0.01f

The maximum time of the update cycle for threads.
Used when the IF_ALL_THREADS_VERY_BUSY policy is selected.
Thus, if no thread in the pool satisfies the time frame when calling getThread() function, then a new free thread is created.
Measured in seconds.
