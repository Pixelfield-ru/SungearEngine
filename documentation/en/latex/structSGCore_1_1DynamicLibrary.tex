\doxysubsection{Detailed Description}
Allows you to load a dynamic library from a local path, load library functions and call them in the future.~\newline
Note that the functions you want to load must be marked with SG\+\_\+\+DLEXPORT and (in most cases) SG\+\_\+\+NOMANGLING to disable function name mangling.
