Creates a standard plugin project.~\newline
~\newline
Creates files in the plugin folder\+: CMake\+Lists.\+txt for building the plugin;~\newline
Sources/\+Plugin\+Main.\+h and Sources/\+Plugin\+Main.\+cpp for loading the plugin by the engine;~\newline
Sources/\{plugin\+Name\}.h and Sources/\{plugin\+Name\}.cpp to implement a structure called \{plugin\+Name\} that inherits \mbox{\hyperlink{structSGCore_1_1IPlugin}{SGCore\+::\+IPlugin}}.

\begin{DoxyNote}{Note}
Please note that to create a plugin project you must have the SUNGEAR\+\_\+\+SOURCES\+\_\+\+ROOT environment variable set.~\newline
It should indicate the path to the Sungear Engine sources.
\end{DoxyNote}
